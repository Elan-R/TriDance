Step 1 — Trust local HTTPS (one-time)

To enable motion sensors & wake lock on iOS (and avoid warnings elsewhere), install and trust the local CA certificate used by this server.

Download CA certificate (.cer) Download CA certificate (.pem)

iPhone / iPad (iOS / iPadOS)
  1. Tap Download CA certificate above (prefer .cer). If prompted, allow download.
  2. Open Settings → Profile Downloaded (or Settings → General → VPN & Device Management) → Install.
  3. Then go to Settings → General → About → Certificate Trust Settings and enable Full Trust for this certificate.
  4. Return to this page and tap Start.

Tip: If you don’t see “Profile Downloaded”, open the downloaded file from Safari’s downloads list.

Android (Chrome)
  1. Download the .cer above.
  2. Open Settings → Security (or Privacy & Security) → Encryption & credentials → Install a certificate → choose CA certificate.
  3. Select the downloaded file and confirm. You may need a screen lock set.
  4. Reload this page and tap Start.

Note: Some apps ignore user CAs, but Chrome honors them for browsing.

macOS (Safari/Chrome)
  1. Download the .cer file.
  2. Double-click it to open in Keychain Access. Choose login (or System) keychain.
  3. Find the certificate, double-click, set When using this certificate to Always Trust.
  4. Reload this page and tap Start.
Windows (Edge/Chrome)
  1. Download the .cer file.
  2. Double-click → Install Certificate… → choose Local Machine → store: Trusted Root Certification Authorities.
  3. Restart the browser, reload this page, then tap Start.

Only trust certificates you created yourself on a machine you control.
